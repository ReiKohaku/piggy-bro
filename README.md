## 运行环境

Node >= 12

## 调试与安装部署

1. 在任意终端中切换到此文件所在文件夹，如：
   ```shell
   cd /opt/git/piggy-bro
   ```

2. 安装依赖。
   ```shell
   npm i
   ```

3. 启动运行脚本。
   ```shell
   npm run start
   ```
